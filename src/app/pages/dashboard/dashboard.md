# Dashboard Page

The `DashboardComponent` is the main workspace for authenticated users to manage students.

## Features
- **Protected Route**: Only accessible to logged-in users via the `authGuard`.
- **Student Management**: Hosts the `StudentListComponent` to provide full CRUD capabilities.
- **Full-Width Layout**: Designed to utilize the available screen space for managing student records.

## Components Used
- `StudentListComponent`
