# Home Page

The `HomeComponent` serves as the landing page for unauthenticated users.

## Features
- **Hero Section**: Displays a welcoming title and subtitle for the Student Management System.
- **Authentication Tabs**: Allows users to toggle between the Login and Signup forms.
- **Auth Guard**: Automatically redirects logged-in users to the `/dashboard`.
- **Responsive Layout**: Centered content designed to work across various screen sizes.

## Components Used
- `LoginComponent`
- `SignupComponent`
